Tweets and followers

If you want to view the tweets of a given username you can type any name into the textfield above an click on the button. If you want to view the followers of a username you can do the same as before but click the button.

Sentiment Analysis

To analyze the sentiment of any English text you can click on the Sentiment Analysis link in the navbar above. Arrived at the page you can input any text into the textfield. By clicking the button the application will analyze each word and present the overall sentiment as well as the sentiment per word.

Dataset Analysis

To analyze a Twitter dataset you can press on the Dataset link in the navbar above. At the page you can provide a text input to request a dataset from the Twitter API. By clicking the the application will request a dataset of 1000 tweets and the corresponding users. It then analyzes the top 10 hashtags and top 10 users from the dataset.
